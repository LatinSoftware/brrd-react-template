# Plantilla Base de React con Vite

Esta es una plantilla base para iniciar proyectos de React utilizando Vite. Vite es un entorno de desarrollo rápido para JavaScript con React, proporcionando una experiencia de desarrollo veloz y eficiente.

## Características

- Configuración optimizada para desarrollo con Vite.
- Integración preconfigurada con React.
- Soporte para importación de módulos ES6, importación de archivos CSS, y más, sin configuración adicional.
- Configuración de rutas y estructura de carpetas predefinidas para una fácil organización del código.
- Utilización de tailwind para el manejo de estilos
- Integración con libreria de componentes `React Prime`

## Requisitos Previos

- Node.js >= 20 y pnpm instalados en tu máquina.

## Inicio Rápido

1. **Clona el repositorio:**

```bash
  git clone https://github.com/tu-usuario/plantilla-react-vite.git
```
2. **Acceder a la carpeta react-template**
```bash
  cd react-template 
```
3. **Iniciar entorno de desarrollo**

```bash
  pnpm run dev
```
